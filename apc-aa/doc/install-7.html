7. Troubleshooting and Maintenance

7.1 Common problems

There are certain issues with Apache 2 and PHP.  If your web server is Apache 2, make sure that you have the lastest PHP available. At the time of 2.6.0 release, the oficial RedHat distribution contained a a PHP rpm version with the unresolved bug. The only way to get this working is to download latest PHP and create RPM package yourself.

If you experience some other problems, search the mailinglists archives and don't hesitate to contact the community - there is lot's of people who have enough experience with ActionApps installations and most of us have seen various bugs.

7.2 Maintenance

There is no maintenance to do, except that you must watch out always to have enough disk space for all the data. If your cron script is set properly, it optimizes the database according to AA cron setup.
